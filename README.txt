1. 졸업과제 폴더 구성
전체 구성은 졸업 과제 보고서들(착수보고서, 중간보고서, 최종보고서)과 발표자료, 포스터 파일, 시연동영상(기본 연주, 웹 제어) 로 되어 있다.

2. 결과물
다른 파일들은 따로 설명할 필요가 없으나, 결과물은 부연 설명이 필요하다. instruments 폴더는 아두이노 메가로 만든 신디사이저 소프트웨어이다. 아두이노 코드는 확장자로 .ino를 사용하고 시작하는 함수는 setup 이후에 loop로 들어간다. 실행 파일은 harpsichord.ino이다.  middle 폴더에 있는 아두이노 소스는 악기와 웹 사이의 매개자 역할을 하는 아두이노 우노에 들어가는 코드이다. server 폴더에는 웹 서버와 웹 클라이언트 페이지를 구현한 파일들이 있다. 클라이언트 소스는 tmp 폴더의 socketio.html이다. 노드 js로 서버를 실행할 파일은 index.js이다.
